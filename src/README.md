This directory contains the main source code files for the California Housing Price Prediction project.

# 1. Files

## 1.1 Main Scripts
- `ddd.py`: Main training script that implements the XGBoost model training pipeline
- `predict_new_data.py`: Script for making predictions on new data using the trained model

# 2. Code Structure

## 2.1 Training Pipeline (`ddd.py`)
- Data loading and preprocessing
- Model training and validation
- Performance evaluation
- Results visualization

## 2.2 Prediction Pipeline (`predict_new_data.py`)
- Loading trained model
- Processing new data
- Generating predictions
